# Calculadora

Proyecto de calculadora hecha con HTML, CSS y JavaScript.

## Operaciones básicas
- Sumar
- Restar
- Multiplicar
- Dividir

## Funciones avanzadas
- Uso de decimales
- Cálculo con porcentaje (%)
- Cambio de signo (+/-)
- Borrado del último número (←)
- Limpieza total (C)

## Control por teclado
- Números (0-9)
- Operadores (+,-,*,/)
- Enter → calcular resultado
- Backspace → borrar último carácter
- Escape → limpiar pantalla
- Punto (.) → decimales

## Visualización de operaciones
- Muestra la operación en tiempo real
- Muestra el resultado final
- Gestión de errores

## Tecnologías
- HTML
- CSS
- JavaScript
